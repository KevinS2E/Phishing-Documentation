# Spotting phishing

## Introduction

Phishing is an attempt to trick someone by email, message, or a fake website so they click, download something, or share sensitive information.

In a company, this type of attack can put accounts, internal data, and entire systems at risk. That is why it is important to pause before acting.

Attackers often imitate vendors, internal teams, or familiar platforms so the message looks legitimate.

:::tip
The most useful signal is not always a single word or a single detail. It is usually the combination of several small clues.
:::

## Common phishing signals

Review these indicators carefully:

- Spelling or grammar mistakes.
- Suspicious links or links that do not match the visible text.
- Urgent or pressuring requests.
- Requests for credentials, passwords, or codes.
- Strange domains or domains with tiny variations.
- Unknown or unexpected senders.
- Attachments you were not expecting.

:::warning
If an email asks for immediate action, money, passwords, or file downloads, stop before interacting.
:::

## What to do if you receive a suspicious email

1. Do not click links.
2. Do not download attachments.
3. Do not reply to the sender.
4. Do not provide credentials or personal information.
5. Use the "Report Phishing" option available in Outlook.
6. Wait for validation.
7. Delete the message only after reporting it.

Reporting suspicious emails helps protect the whole organization and allows security teams to analyze and block similar threats.

## How to report a suspicious email

![IMAGE - REPORT PHISHING BUTTON IN OUTLOOK](/images/Phishing-Alert-Button.png)

*IMAGE - REPORT PHISHING BUTTON IN OUTLOOK*

### Basic steps

- Select the suspicious email.
- Use the **Report Phishing** button in Outlook.
- Send the message to the security review queue.
- Wait for the response before deleting or archiving it.

### After reporting

The security team can review the message, look for related patterns, and block similar campaigns for other users.

### Why report it even if you are not sure

You do not need absolute certainty. If something looks unusual, reporting it is the best move because it lets specialists review the case.

:::tip
If you are unsure about an email, it is better to report it. The security team can analyze it and determine whether it is a risk.
:::

:::warning
Never interact with links, attachments, or credential requests before reporting the message.
:::
